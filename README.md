Excel Data Analysis & KPI Dashboard

A hands-on data analysis project built entirely in Microsoft Excel, demonstrating end-to-end data processing from raw data cleaning through to an interactive KPI dashboard with dynamic filtering.


Project Overview

This project showcases core Excel data analysis skills applied to a real dataset, covering data preparation, analysis, and interactive visualisation — all within a single Excel workbook.


What Was Done

1. Data Cleaning
- Removed duplicate records and blank rows to ensure data integrity
- Standardised inconsistent formatting across columns (dates, text, numbers)
- Handled missing values and corrected data entry errors
- Ensured all data types were consistent and ready for analysis

2. Lookup Functions
- Used **VLOOKUP** and **XLOOKUP** to cross-reference and merge data from multiple sheets
- Applied **INDEX MATCH** for flexible, dynamic lookups across large datasets
- Used **IF** and nested logical functions to flag and categorise records

3. Pivot Tables
- Built Pivot Tables to summarise and aggregate data by key dimensions
- Analysed trends across categories, time periods, and segments
- Used calculated fields to derive additional metrics directly within the Pivot Table

4. KPI Dashboard
- Designed a clean, interactive KPI dashboard to visualise key metrics at a glance
- Included KPI cards showing headline figures (totals, averages, growth %)
- Used charts (bar, line, pie) linked to Pivot Tables for dynamic updates

5. Slicers & Timelines
- Added **Slicers** to allow one-click filtering by category, region, or segment
- Integrated **Timelines** for date-based filtering by month, quarter, or year
- Connected slicers and timelines across multiple Pivot Tables for a seamless dashboard experience
